Creating a React Component

In React, components are the building blocks of the user interface. A component is simply a JavaScript function that returns JSX. Letâ€™s start by creating our first component:


<script type="text/babel">
    function Header() {
        return (
            <h1>Eve's Kitchen</h1>
        );
    }

    ReactDOM.render(<Header>, document.getElementById('root'));
</script>

Once we create our first component, Header, we use the ReactDOM.render() function to render the component to the DOM. Now, let's add another component:


<script type="text/babel">
    function Main() {
        return (
            <section>
                <p>We serve the most delicious food around.</p>
            </section>
        );
    }

    ReactDOM.render(<Main>, document.getElementById('root'));
</script>

When creating components, always remember to use capital letters for the component names and return JSX. You can combine multiple components into a parent component, as shown below:


<script type="text/babel">
    function App() {
        return (
            <div>
                <Header>
                <Main>
            </div>
        );
    }

    ReactDOM.render(<App>, document.getElementById('root'));
</script>

In this example, we created an App component that renders both the Header and Main components. Always remember that React requires adjacent JSX elements to be wrapped in an enclosing tag, such as a <div>.
