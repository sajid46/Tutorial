Creating React Elements

We are using the ReactDOM.render() function to take the createElement() call that we added and inject it into the document at the element called root. We are telling React to find that element on the page and render our React code inside it.

In React, we think of elements as little bits of UI that can be rendered at any time. Instead of nesting functions inside other functions, let’s create a variable for the element.

Here’s the modified code where we assign the React element to a variable:


<script type="text/javascript">
    const heading = React.createElement('h1', null, 'Getting Started with React!!!');
    ReactDOM.render(heading, document.getElementById('root'));
</script>

This creates a React element and assigns it to the heading variable. Now, React can render this element at any time. We also added a couple of exclamation marks to confirm it works. And it does!
