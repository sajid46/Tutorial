Working with JSX Syntax

JSX in React allows us to inject dynamic content into our elements simply by referencing them with variable names. Let's explore how this works.

We can start by creating variables for the content we want to inject:

    ReactDOM.render(list, document.getElementById('root'));

In this example, we are creating variables for robot, cowboy, and moon emojis. We are also using a variable name for the text react, and we are transforming it to uppercase using name.toUpperCase().

JSX allows us to dynamically inject content into tags using curly braces. Additionally, we can run JavaScript functions, such as toUpperCase() or name.length, inside the curly braces to manipulate the data displayed in the UI.
